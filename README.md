user-management
===============

Meteor user management pages and methods for applications requiring user permissions with hierarchy.

Basic pages to manage 3 user types: 
  admin, 
  manager, 
  staff. 

Includes pages for: 
  user profiles, 
  user management, 
  admin settings, 
  manager settings. 
  
Relies on:
  alanning:roles,
  iron:router.
